# Radix.AVAX
Una solución Web3 en Avalanche que transforma las cadenas de suministro agrícolas mediante pasaportes digitales en NFT. Protege la privacidad financiera del productor con tokens eERC20 y automatiza pagos instantáneos con flujos nativos x402. Transparencia, comercio justo y cumplimiento ambiental sin intermediarios abusivos.
